# 高校公寓管理系统后台


项目简介：
- 前端采用Vue.js+Element UI框架
- 后端采用Spring Boot+MyBatis+Shiro框架，接口风格为Restful风格
- 数据库使用MySql
- 前端运行在nginx服务器中，后端运行在Tomcat服务器中
